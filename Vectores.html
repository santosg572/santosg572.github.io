  1. Find the general solution of the differential equation x’ = ax + 3 where a is a parameter. What are the equilibrium points for this equation? For which values of a are the equilibria sinks? For which are they sources?

  2. For each of the following differential equations, find all equilibrium solutions and determine if they are sinks, sources, or neither. Also, sketch the phase line.

    1. x′ = x3 − 3x

    2. x′ = x4 − x2

    3. x′ = cos x

    4. x′ = sin2x

    5. x′ = |1 − x2|

  3. Each of the following families of differential equations depends on a parameter a. Sketch the corresponding bifurcation diagrams.

    1. x′ = x2 − ax

    2. x′ = x3 − ax

    3. x′ = x3 − x + a

  4. Consider the function f(x) whose graph is displayed in Figure 1.12.

    1. Sketch the phase line corresponding to the differential equation x’ = f(x).

    2. Let ga(x) = f(x) + a. Sketch the bifurcation diagram corresponding to the family of differential equations x′ = ga(x).

    3. Describe the different bifurcations that occur in this family.

  5. Consider the family of differential equations


    x′ = ax + sin x

    where a is a parameter.

    1. Sketch the phase line when a = 0.

    2. Use the graphs of ax and sin x to determine the qualitative behavior of all of the bifurcations that occur as a increases from -1 to 1.

    3. Sketch the bifurcation diagram for this family of differential equations.

  6. Find the general solution of the logistic differential equation with constant harvesting


    x′ = x(1 − x) − h

    for all values of the parameter h > 0.

    1. Find a solution of this equation satisfying x(0) = 4. Describe the qualitative behavior of this solution.

    2. Find a solution of this equation satisfying x(0) = 3. Describe the qualitative behavior of this solution.

    3. Describe the qualitative behavior of any solution of this system as t → ∞.

  8. Consider a first-order linear equation of the form x’= ax + f (t) where a ∈ R. Let y(t ) be any solution of this equation. Prove that the general solution is y(t ) + c exp(at ) where c ∈ R is arbitrary.

  9. Consider a first-order, linear, nonautonomous equation of the form x’(t) = a(t )x.

    1. Find a formula involving integrals for the solution of this system.

    2. Prove that your formula gives the general solution of this system.

  10. Consider the differential equation x’ = x + cos t .

    1. Find the general solution of this equation.

    2. Prove that there is a unique periodic solution for this equation.

    3. Compute the Poincaré map p : t = 0 → t = 2π for this equation and use this to verify again that there is a unique periodic solution.

  11. First-order differential equations need not have solutions that are defined for all times.

    1. Find the general solution of the equation x = x2.

    2. Discuss the domains over which each solution is defined.

    3. Give an example of a differential equation for which the solution satisfying x(0) = 0 is defined only for  − 1 < t < 1.

  12. First-order differential equations need not have unique solutions satisfying a given initial condition.

    1. Prove that there are infinitely many different solutions of the differential equations x′ = x1/3 satisfying x(0) = 0.

    2. Discuss the corresponding situation that occurs for x’ = x/t , x(0) = x0.

    3. Discuss the situation that occurs for x′ = x/t2, x(0) = 0.

  13. Let x’ = f(x) be an autonomous first-order differential equation with an equilibrium point at x0.

    1. Suppose f′(x0) = 0. What can you say about the behavior of solutions near x0? Give examples.

    2. Suppose f(x0) = 0 and f″(x0) ≠ 0. What can you now say?

    3. Suppose f(x0) = f″(x0) = 0 but f(x0) ≠ 0. What can you now say?

  14. Consider the first-order nonautonomous equation x’ = p(t )x where p(t ) is differentiable and periodic with period T. Prove that all solutions of this equation are periodic with period T if and only if


    ∫0Tp(s)ds = 0

    f(t, p) > 0, f(t, q) < 0
